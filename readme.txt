The ZgBoke Alliance Navigation Wordpress Plugin. 中国博客联盟成员展示页面插件

== Description ==

Quickly create a navigation page, showing members of the Blogs alliance.

快速创建一个博客导航页面，展示中国博客联盟成员。

插件特色    
    这个插件是一个微型导航，用于展示中国博客联盟(zgboke.com)的成员    
    1、成员数据与中国博客联盟官网实时同步，拥有公平的排名机制；    
    2、页面采用JS封装，杜绝任何外链输出，SEO无后顾之忧;；    
    3、样式精致耐看，响应式设计，最低支持225px的超窄宽度。    
        
博客联盟简介    
    中国博客联盟(zgboke.com)于2014年6月由草根博主张戈创立，收录国内各个领域的优秀博客，这是一个全人工编辑的开放式博客联盟交流和展示平台。联盟成员完全平等、自由！无论你是技高一人，还是文采飞扬，还是百万家产，在这里，我们都是平等的，这里没有庸俗的歧视和奉承，这里只有自由而平等的对话，只有团结与互助。
    这里支持高质量的博客，支持原创内容，反对采集与作弊！联盟提倡互推互访，共同进步！

== Installation ==

可以通过以下两种方法的其中一种来安装ZgBoke-Nav 插件：

1. 将下载的文件解压缩，然后将`zgboke-nav`文件夹 上传到 `/wp-content/plugins/`目录，在插件后台启用

2. 直接在后台-安装插件，搜索'zgboke-nav'，按照提示安装启用

Ps: 首次启用插件，请进入设置界面进行更详细的喜好设置
